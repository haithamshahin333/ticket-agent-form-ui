Ticket Support Agent

Enter a ticket ID and a message, then Run. The agent looks up the ticket and replies. Conversation continues in one thread until you start a new one. Try TICKET-101 or TICKET-102.

Deployment auth

Blank for local langgraph dev (no auth). Required when this page runs on a LangSmith deployment — the Runs API is gated by x-api-key. Kept in memory for this tab only (not stored). Don't use on a page shared with untrusted users.
